# 🎬 Movie FTP Server

A beautiful web interface to browse, stream, and download movies from an FTP server.

## Features

- 📂 Browse movies by category (Action, Comedy, Drama, etc.)
- 🔍 Search movies across all categories
- ▶️ Stream movies directly in browser
- ⬇️ Download movies with one click
- 📱 Fully responsive mobile-friendly design
- 🚀 Fast and lightweight
